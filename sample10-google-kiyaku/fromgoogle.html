HTML

私とは違ったルールを、改めるためにアウトプット

  • <script src="https://ajax.googleapis.com"> httpsを指定
  • <script src="://ajax.googleapis.com"> のように省略して指定しない
  • h1, h2, h3等は順番がゴッチャでも大丈夫
  • liタグは閉じなくてよい
  • thead, tbody, tr, th, tdタグも同じ
  • class属性を最優先
  • id属性は出来るだけ使用を避ける

CSS

  • .クラス名の使用を推奨。
  • li.クラス名等の使い方をしない。パフォーマンスに影響。
  • font, padding, margin等々、一遍に指定できるのは指定する
  • !importantの使用は避ける
  • marginやpaddingなどの指定で、少数（-1 < x < 1）の時、.5emとか書いていたが、0.5emと0も付ける
